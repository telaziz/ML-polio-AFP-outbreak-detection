# ML-polio-AFP-outbreak-detection
# Machine Learning Framework for Early Detection of Polio Outbreaks from AFP Surveillance Data

![Polio Antenna Logo](https://www.polioantenna.org/assets/logo.png)  
**Preprint** | Authors: Honey Gemechu, Gelan Ayana, Jude Kong et al. | SSRN: https://ssrn.com/abstract=5797346

## Overview
This repository hosts the preprint research paper developing a CatBoost-based ML model for predicting polio outbreaks using real-world Acute Flaccid Paralysis (AFP) surveillance data from Ethiopia (2017–2022).

**Key Results**:
- Highest performance: CatBoost with **92.22% accuracy** and **AUC 0.99**
- Top predictors: Body temperature, fatigue, sore throat
- Contribution to **Global Polio Eradication Initiative (GPEI)** and **WHO** AFP surveillance enhancement

This work is linked to the [Polio Antenna Project](https://polioantenna.org/) – an AI-driven initiative for community-based polio surveillance in Ethiopia.
